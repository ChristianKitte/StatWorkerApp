Über uns...


Diese Webseite entstand im Rahmen des standortübergreifenden Moduls Objektorientierte Skriptsprachenals der FH Brandenburg unter Leitung von Prof. Dr. Thomas Preuss im SS18. Ziel war die Erstellung einer im Kern auf dem Framework Django beruhenden Webseite zur Berechnung und grafischer Aufbereitung statistischer Werte für frei eingebbare Daten.

Clientseitig handelt es sich bei dieser Anwendung um eine Single Page Application. Hierbei wird zu Anfang nur eine Seite geladen, weiterer Content jedoch dynamisch aktualisiert und nachgeladen. Als Techniken und Sprechen kommen HTML5, JavaScript, CSS sowie JQuery und Bootstrap zum Einsatz.

Serverseitig kommt das auf Python basierende Framework Python mit einigen weiteren Paketen zum Einsatz. Besonders möchten wir hier auf Seaborn zur Erzeugung von Grafiken und OpenPyXL zum Zugriff auf Excel hinweisen.
